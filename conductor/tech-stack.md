# HomeMade - Tech Stack

## Core Technologies
- **Framework:** **Next.js 14+ (App Router)** - Leveraging the latest Next.js features for performance and developer experience.
- **Language:** **TypeScript** - Strict typing across the codebase for better maintainability and error detection.
- **Styling:** **TailwindCSS 4** - Modern, utility-first CSS framework for efficient and consistent design.
- **Backend & Auth:** **Supabase** - Using @supabase/ssr and @supabase/supabase-js for real-time data sync and secure authentication.
- **Testing:** **Vitest** - Fast, modern test runner with React Testing Library and Happy-Dom for reliable UI validation.

## Specialized Integrations
- **AI Engine:** **Google Generative AI (Gemini)** - Powering the AI recipe generation with robust fallback mechanisms and error handling.
- **Localization:** Native **Hebrew (RTL)** support - Integrated into the layout and UI components.

## Development & Deployment
- **State Management:** **React Hooks (useState/useEffect)** combined with Supabase for persistent and transient state.
- **Linting:** **ESLint** - Following Next.js recommended configurations.
- **Deployment Platform:** **Vercel** - Optimized for Next.js applications.

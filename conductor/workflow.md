# HomeMade - Workflow

## Development Process
1. **Scaffold:** Initialize Next.js, Tailwind, and project structure.
2. **Infrastructure:** Set up Supabase project, database schema, and Auth.
3. **Core - Pantry:** Implement ingredient CRUD and categorization.
4. **Core - Recipes:** Implement recipe discovery UI and mock logic.
5. **History:** Implement usage tracking.
6. **Polishing:** UX refinements and final deployment to Vercel.

## Standards
- Follow standard Next.js App Router conventions.
- Use TypeScript for all components and logic.
- Ensure mobile-first responsive design.

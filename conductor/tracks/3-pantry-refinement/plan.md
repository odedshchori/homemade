# Track 3: Pantry Refinement

## Goal
Improve the pantry management experience by adding editing capabilities and refining the UI.

## Tasks
1. [x] Create a modal or inline edit form for updating ingredient details (name, category, unit).
2. [x] Add quick "plus" and "minus" buttons for adjusting quantity without opening an editor.
3. [x] Improve visual feedback when adding or deleting items.
4. [x] Implement search and filtering (by category) to handle larger pantries.

## Verification
- Can edit existing ingredient names and categories.
- Can quickly update quantity using +/- buttons.
- Search and filters correctly narrow down the ingredient list.
